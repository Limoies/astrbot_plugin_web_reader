## 🚧 计划中的功能

### 功能增强
- [ ] **批量读取** — 一次性读取多个 URL：`/readweb url1 url2 url3`
- [ ] **定时任务** — 利用 AstrBot 的 cron 机制定时抓取：`/readweb --cron "0 8 * * *" https://news.com`
- [ ] **更多文件格式** — .epub / .pptx / .txt 本地文件支持
- [ ] **预览增强** — 在聊天中直接展示结构化摘要，而不是只给文件路径
- [ ] **输出长度控制** — 超长内容自动分段，避免消息被截断

### CDP 相关
- [ ] **自动检测 Chrome 状态** — 在 CDP 模式前自动检测 Chrome 是否已启动调试模式
- [ ] **自动启动 Chrome** — 如果检测到 Chrome 未启动调试模式，自动帮忙启动

### 可用性
- [ ] **进度反馈** — 读取大文件时显示进度百分比
- [ ] **取消操作** — 支持取消正在进行的读取任务
- [ ] **缓存机制** — 短时间内重复读取同一 URL 时使用缓存
